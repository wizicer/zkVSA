Proof Verification

Load proof data from the gnark-circuit project and verify it using the Go WASM module.

Instructions

  1. Click "Load WASM Module" to load the Go WASM verifier (13MB download)
  2. Run the gnark-circuit project to generate proof data: cd gnark-circuit && go run .
  3. Click "Load Proof Data" to load the generated proof, or copy the proof_data.json content into the textarea
  4. Click "Verify Proof" to verify the proof using the Go WASM module

Note: The WASM module is loaded on-demand to improve initial page load time.
